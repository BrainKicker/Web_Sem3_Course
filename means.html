Средние значения

Пусть дана функция \(\text{M}(X)\), где \(X\) — непустой массив неотрицательных чисел длины \(N\) (\(X = [x_1, x_2, ..., x_N], \ x_i \in \mathbb{R}_+, \ N > 0\)). На вход функции \(\text{M}(X)\) может поступать массив любой длины (то есть \(N\) тоже является переменной).

\(\text{M}: \mathbb{R}_+^N \rightarrow \mathbb{R}_+\)

Функция \(\text{M}(X)\) обладает следующими свойствами:

  1. Значение функции не зависит от порядка элементов в массиве:

    \(\text{M}(x_1, x_2, ..., x_N) = \text{M}(x_{i_1}, x_{i_2}, ..., x_{i_N}), \ i_j \in \overline{1,N}, \ i_j \ne i_k \ | \ j \ne k\)

  2. Значение функции находится между минимальным и максимальным элементами массива:

    \(\min{X} \leq \text{M}(X) \leq \max{X}\)

  3. Значения функции от массива \(X\) и от массива \(X \cup [\text{M}(X)]\) совпадают:

    \(M(x_1, x_2, ..., x_N) = \text{M}(x_1, x_2, ..., x_N, \text{M}(X))\)

  4. Функция \(\text{M}(X)\) монотонно не убывает относительно каждого элемента массива:

    \(\DeclareMathOperator{\dom}{dom} \forall \ a, b \in \dom{\text{M}}: a < b \Rightarrow \text{M}(..., x_i = a, ...) \leq \text{M}(..., x_i = b, ...), \ i \in \overline{1,N}\)

  5. Если \(i\)-тый элемент массива меньше значения функции, то при его уменьшении он будет меньше нового значения функции. Если \(i\)-тый элемент массива больше значения функции, то при его увеличении он будет больше нового значения функции:

    \(x_i < \text{M}(X) \Rightarrow x_i - \epsilon < \text{M}(..., x_i - \epsilon, ...), \ i \in \overline{1,N}, \ \epsilon > 0\)

    \(x_i > \text{M}(X) \Rightarrow x_i + \epsilon > \text{M}(..., x_i + \epsilon, ...), \ i \in \overline{1,N}, \ \epsilon > 0\)

Функцию, обладающую всеми этими свойствами, мы будем называть средним значением.

Основные средние значения:

  • Среднее арифметическое.

    \(\frac{\sum_{i=1}^N{x_i}}{N}\)

    Доказательство свойств:

    1. Элементы массива фигурируют только в сумме, а суммирование — коммутативная операция, следовательно, её значение не зависит от порядка элементов. Значит, и значение всего выражения не от него не зависит.

    2. Сравним три значения: \(\min{X}, \frac{\sum_{i=1}^N{x_i}}{N}, \max{X}\).

      Умножим все три выражения на \(N\):

      \(\min{X} \times N, \sum_{i=1}^N{x_i}, \max{X} \times N\)

      В сумме \(N\) слагаемых, каждое из которых больше либо равно минимальному элементу и меньше либо равно максимальному, следовательно вся сумма больше либо равна \(N\)-ному количеству минимальных элементов и меньше либо равна \(N\)-ному количеству максимальных элементов:

      \(\min{X} \times N \leq \sum_{i=1}^N{x_i} \leq \max{X} \times N\)

      Делим каждое выражение на \(N\) и получаем:

      \(\min{X} \leq \frac{\sum_{i=1}^N{x_i}}{N} \leq \max{X}\)

    3. \(\text{M}(X \cup \text{M}(X)) = \large \frac{\sum_{i=1}^N{x_i} + \frac{\sum_{i=1}^N{x_i}}{N}}{N+1} \normalsize = \large \frac{N \times \sum_{i=1}^N{x_i} + N \times \frac{\sum_{i=1}^N{x_i}}{N}}{N(N+1)} \normalsize = \large \frac{N \times \sum_{i=1}^N{x_i} + \sum_{i=1}^N{x_i}}{N(N+1)} \normalsize = \large \frac{(N+1) \times \sum_{i=1}^N{x_i}}{N(N+1)} \normalsize = \large \frac{\sum_{i=1}^N{x_i}}{N} \normalsize = \text{M}(X)\)

    4. \(\large \frac{\partial \text{M}}{\partial x_i} \normalsize = \large \frac1N \normalsize, \ i \in [1,N]; \ N > 0 \Rightarrow \large \frac1N \normalsize > 0\)

      Производная всегда больше нуля — значит она неотрицательна.

      Производная существует и неотрицательна на всей области определения, значит функция монотонно не убывает относительно элемента \(x_i\).

    5. \(\sqsupset x_i < \text{M}(X) \Leftrightarrow \text{M}(X) - x_i = \xi > 0, \ i \in [1,N]; \ \epsilon > 0:\)

      \(\text{M}(..., x_i - \epsilon, ...) = \large \frac{\sum_{j=1}^{i-1}{x_j} + (x_i - \epsilon) + \sum_{j=i+1}^N{x_j}}{N} = \frac{\sum_{i=1}^N{x_i} - \epsilon}{N} = \frac{\sum_{i=1}^N{x_i}}{N} - \frac{\epsilon}{N} = \normalsize \text{M}(X) - \large \frac{\epsilon}{N} \normalsize\)

      \(\text{M}(..., x_i - \epsilon, ...) - (x_i - \epsilon) = \normalsize \text{M}(X) - \large \frac{\epsilon}{N} \normalsize - x_i + \epsilon = (\text{M}(X) - x_i) + (\epsilon - \large \frac{\epsilon}{N} \normalsize) = \xi + \large \frac{N-1}{N} \normalsize \times \epsilon > 0\)

      Значит, \(x_i - \epsilon < \text{M}(..., x_i - \epsilon, ...)\).

      Аналогичные рассуждения проводятся для случая \(x_i > \text{M}(X)\).

  • Среднее геометрическое.

    \(\sqrt[N]{\prod_{i=1}^N{x_i}}\)

    Доказательство свойств:

    1. Элементы массива фигурируют только в произведении, а это коммутативная операция, следовательно, её значение не зависит от порядка элементов. Значит, и значение всего выражения не от него не зависит.

    2. Сравним три значения: \(\min{X}, \sqrt[N]{\prod_{i=1}^N{x_i}}, \max{X}\).

      Возведём все три выражения в \(N\)-ную степень:

      \(\left(\min{X}\right)^N, \prod_{i=1}^N{x_i}, \left(\max{X}\right)^N\)

      В левом произведении \(N\) множителей, каждое из которых меньше либо равно каждому множителю в произведении по центру, каждый из которых в свою очередь меньше либо равен каждому множителю в произведении справа.

      Следовательно:

      \(\left(\min{X}\right)^N \leq \prod_{i=1}^N{x_i} \leq \left(\max{X}\right)^N\)

      Возводим все три выражения в степень \(\frac1N\) и получаем:

      \(\min{X} \leq \sqrt[N]{\prod_{i=1}^N{x_i}} \leq \max{X}\)

    3. \(\text{M}(X \cup \text{M}(X)) = \sqrt[N+1]{\prod_{i=1}^N{x_i} \times \sqrt[N]{\prod_{i=1}^N{x_i}}} = \sqrt[N+1]{\prod_{i=1}^N{x_i} \times \prod_{i=1}^N{x_i^{\frac1N}}} = \sqrt[N+1]{\prod_{i=1}^N{x_i^{1+\frac1N}}} =\)

      \(= \sqrt[N+1]{\prod_{i=1}^N{x_i^{\frac{N+1}{N}}}} = \prod_{i=1}^N{\left(x_i^{\frac{N+1}{N}}\right)^{\frac{1}{N+1}}} = \prod_{i=1}^N{x_i^{\frac1N}} = \sqrt[N]{\prod_{i=1}^N{x_i}} = \text{M}(X)\)

  • Среднее гармоническое.

    \(\frac{N}{\sum_{i=1}^N{\frac{1}{x_i}}}\)

  • Среднее квадратическое.

    \(\sqrt{\frac{\sum_{i=1}^N{x_i^2}}{N}}\)
